### 步骤 2：创建 GitHub Actions 工作流

**文件：`.github/workflows/news-fetch.yml`**

用途：每日固定 2 轮抓取（北京 9:00 / 21:00 = UTC 1:00 / 13:00）。

内容要点：
- `name: 知讯数据抓取`
- `on.schedule.cron`：`"0 1 * * *"` 和 `"0 13 * * *"`（UTC）
- `on.workflow_dispatch`：支持手动触发
- jobs 步骤：
  1. `actions/checkout@v4`
  2. `actions/setup-python@v5`（python-version: "3.11.9"）
  3. 安装 uv（`astral-sh/setup-uv@v3`）
  4. `uv sync`（在 news-pipeline/ 目录）
  5. 运行管线（`uv run python -m news_pipeline`，env 注入 `NEWS_LLM_API_KEY: ${{ secrets.NEWS_LLM_API_KEY }}`）
  6. 提交并推送 `news-pipeline/data/news.json`（`git config` + `git commit` + `git push`，使用 `github-actions[bot]` 身份）
- 权限：`contents: write`
